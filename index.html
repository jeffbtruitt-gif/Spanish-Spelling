📅 How to update the list each week

  1. Open index.html on GitHub, click the pencil ✏️ to edit
  2. Find the GITHUB_LISTS section (clearly marked with a comment)
  3. Copy/paste a new week block, fill in the words
  4. Click Commit — done in about a minute ✅

Every device (phone, tablet, laptop) will see the new list automatically once you commit — no app updates needed.

➕ Create New Spelling List

Use this to add a device-only list for testing. To share a list across all devices, edit index.html on GitHub instead.

Each line becomes one item the student spells. Phrases with spaces are fully supported.

Click Set Active to make a list the current one. Click Edit to change its name or words.
